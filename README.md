# GeoSci.xyz Case Histories

case-histories.geosci.xyz

A collection of case histories in geophysics

We highlight case histories that follow the seven step framework in geophysics:

1. Setup
2. Properties
3. Survey
4. Data
5. Processing
6. Interpretation 
7. Synthesis
